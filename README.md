# Life AI

Life AI is an AI-powered web application built using Spring AI, Image Models, and Angular. The application provides multiple AI-driven utilities focused on creativity and career enhancement.

## Features

### Resume Analyser

The Resume Analyser allows users to upload resume PDFs and receive AI-generated feedback. The system analyzes the resume and provides:

- Resume improvement suggestions
- Career growth recommendations
- Skills and technologies to learn
- Structured career guidance
- Productivity and atomic habit suggestions

### Ghibli Image Creator

The Ghibli Image Creator generates AI images inspired by Studio Ghibli-style aesthetics. Users can generate artistic images using prompts and download the generated outputs.

### AI Wallpaper Generator

The Wallpaper Generator creates personalized wallpapers based on user interests such as:

- Favourite movies
- Anime
- Games
- Places
- Colour themes

The generated wallpapers are AI-powered and optimized for multiple aspect ratios.

## Technologies Used

- Spring Boot
- Spring AI
- Angular
- OpenAI Image Model
- REST APIs

## AI Capabilities

The application integrates AI features including:

- PDF understanding and analysis
- AI image generation
- Prompt-based personalized content generation
- Dynamic HTML response rendering

## Purpose

The goal of Life AI is to combine productivity, creativity, and AI-powered assistance into a single application that helps users improve their careers while also providing creative AI tools.
